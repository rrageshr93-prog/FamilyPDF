# FamilyPDF

> **Free · Open Source · No Trackers**  
> Made with ❤️ by RR for the family

A simple, fully offline PDF editor built for my family.

### Features
- View PDFs
- Fill forms
- Sign documents (draw signature)
- Merge & Split PDFs
- Compress PDFs
- Annotate & Highlight
- Rotate, reorder, delete pages
- OCR (in F-Droid version)
- 100% private - No internet, no trackers
